Project information

  • Category: Data engineering
  • Technology: Azure Data Factory
  • Project: Build a real-world data pipeline in Azure Data Factory (ADF) using Covid19 data.

Following items learned and implemented as part of this project

  • Azure Data Factory
    • Designing a solution architecture for data engineering utilizing Azure Data Engineering technologies including Azure Data Factory (ADF), Azure Data Lake Gen2, Azure Blob Storage, Azure SQL Database, Azure Databricks, Azure HDInsight, and Microsoft PowerBI.
    • Integrating data from various sources such as HTTP clients, Azure Blob Storage, and Azure Data Lake Gen2 using Azure Data Factory.
    • Implementing branching and chaining activities in Azure Data Factory (ADF) Pipelines using control flow activities like Get Metadata, If Condition, ForEach, Delete, and Validation.
    • Employing Parameters and Variables in Pipelines, Datasets, and LinkedServices to create metadata-driven pipelines in Azure Data Factory (ADF).
    • Debugging data pipelines and resolving issues.
    • Debugging data flows, investigating issues, and fixing failures.
    • Implementing Azure Data Factory pipelines to invoke Mapping Data Flows and executing them.
    • Creating ADF pipelines to execute HDInsight activities for data transformations.
    • Creating ADF pipelines to execute Databricks Notebook activities for transformations.
    • Establishing dependencies between pipelines to orchestrate the data flow.
    • Establishing dependencies between triggers to orchestrate the data flow.
    • Monitoring data pipelines, creating alerts, and reporting metrics from the Azure Data Factory Monitor.
    • Monitoring Data Factory pipelines using Azure Monitor and setting diagnostic settings to be forwarded to Azure Storage Account or Log Analytics Workspace.
    • Creating Log Analytics workspace, creating workbooks, and charts from log analytics on the Azure Data Factory pipelines.
    • Implementing the Azure Data Factory Analytics monitoring tool and extending its capabilities further.
  • Azure HDInsight & Databricks
    • Establishing HDInsight Clusters, Navigating the UI, Utilizing Ambari, Formulating Hive tables, Activating HDInsight activities through Azure Data Factory.
    • Setting up Azure Databricks Workspace, Forming Databricks clusters, Linking storage accounts, Drafting Databricks notebooks, executing transformations via Databricks notebooks, Incorporating Databricks notebooks into Azure Data Factory workflows.
  • Azure Devops (CI/CD)
    • Establishing an Azure DevOps Environment and setting up an Azure DevOps Git Repository.
    • Developing a CI/CD process to deploy Azure Data Factory artifacts to elevated environments.
    • Crafting build and release pipelines within Azure DevOps for deploying code to advanced environments such as Testing and Production.
    • Configuring and parameterizing CI/CD pipelines to deploy ADF pipelines with access to Azure Data Lake Storage.
