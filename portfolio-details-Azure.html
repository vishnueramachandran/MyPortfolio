Project information

  • Category: Cloud
  • Technology: Azure
  • Project: Build readit website, a fictional bookstore that's built in Azure.

Following items learned and implemented as part of this project

    • Introduction to Microsoft Azure: Understand the fundamentals of Microsoft Azure, including its features, benefits, and how it fits into the cloud computing landscape.
    • Setting Up Azure Environment: Learn how to set up and configure an Azure environment, including creating an Azure account, navigating the Azure portal, and understanding different Azure services.
    • Azure Services Overview: Explore various Azure services such as Azure Virtual Machines, Azure Storage, Azure App Services, Azure Functions, Azure SQL Database, Azure DevOps, and more.
    • Deployment and Management: Understand the process of deploying and managing resources in Azure, including virtual machines, storage accounts, databases, and web applications.
    • Networking in Azure: Gain knowledge about networking concepts in Azure, including virtual networks, subnets, network security groups, and Azure Load Balancer.
    • Security and Compliance: Learn about security best practices in Azure, including identity and access management, encryption, and compliance standards.
    • Monitoring and Logging: Explore monitoring and logging capabilities in Azure, including Azure Monitor, Azure Log Analytics, and Azure Application Insights.
    • Scalability and High Availability: Understand how to design and implement scalable and highly available solutions in Azure using features such as auto-scaling, availability sets, and Azure Traffic Manager.
    • Cost Management: Learn strategies for managing costs in Azure, including cost analysis, budgeting, and resource tagging.
    • Integration and Automation: Explore how to integrate Azure services with other platforms and automate common tasks using Azure Automation, Azure Logic Apps, and Azure Functions.
